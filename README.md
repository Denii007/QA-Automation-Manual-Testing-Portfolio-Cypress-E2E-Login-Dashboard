# Cypress E2E Login & Dashboard

This repository contains QA Automation testing using Cypress
and manual test case documentation for a React application.

## Tech Stack
- Cypress
- JavaScript
- Node.js

## Manual Test Case
Manual test cases are documented in the following file:
- **React Gallery-Test Cases.pdf**

## Automation Test Coverage
- Login feature (positive & negative scenarios)
- Dashboard access and logout validation

## Prerequisite
Before running the automation tests, make sure the application under test
##is running locally on:
##http://localhost:3000

## Example:
-npm start

##How to Run Automation Test
- npm install
- npx cypress open
